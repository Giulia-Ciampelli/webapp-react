## React movie app
Una web app per cercare film e recensioni.

*Pacchetti*
- react-router-dom
- fontawesome-icons
- ldrs

*Componenti*
- Header
- MovieCard
- Footer

*Pagine*
- Homepage
- Pagina lista film
- Pagina dettaglio film con recensioni

*Contexts*
- Chiamate API
- Formattatore date

*Ricorda*
Da usare con il backend nella repo webapp-express (CORS già installato)
